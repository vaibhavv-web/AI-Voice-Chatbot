# AI Voice Chatbot 🎙️

An AI-powered voice chatbot built using **Llama 3, Whisper, Gradio, and OpenRouter**.

The system allows users to speak to an AI assistant and receive voice responses.

## Features

- Voice input using microphone
- Speech-to-text using Whisper
- LLM responses using Llama 3 via OpenRouter
- Text-to-speech using gTTS
- Interactive UI with Gradio

## Tech Stack

- Python
- Gradio
- OpenRouter API
- Llama 3
- Whisper
- gTTS
